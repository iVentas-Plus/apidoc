Documentación de API

Estadísticas de Broadcast

Endpoint

GET https://restservices.iventas.com/v2/broadcast/stats/:id

Descripción: Recupera las estadísticas de un broadcast específico.

Parámetros

  • :id (string, requerido) - El ID único del broadcast, devuelto por la petición de envío de plantillas.

Encabezados

  • Authorization (string, requerido) - Token Bearer para autenticación. Ejemplo: Bearer iventas-apiv2-xxxxxxxx

Ejemplo de Solicitud

curl --request GET \
--url https://restservices.iventas.com/v2/broadcast/stats/66d236af24aa9a00089ba6c7 \
--header 'Authorization: Bearer iventas-apiv2-03430ff2-4957-47d7-8a7e-a8df982cf1e9'

Respuesta

En caso de éxito, la API devuelve un objeto JSON con las estadísticas del broadcast.

Descripción de los campos:

  • deliveredMessages: Lista de números a los que se confirmó la entrega del mensaje.
  • viewedMessages: Lista de números que visualizaron el mensaje.
